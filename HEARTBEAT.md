# Heartbeat (Elvis / Tota)

## Regla #1 (anti-spam)

Si NO hay una acción clara con ROI o algo urgente: responde HEARTBEAT_OK.

**PAUSA:** No enviar planes ni sugerencias proactivas hasta nueva orden del usuario.
**REACTIVACIÓN:** Solo se reanudan planes/sugerencias cuando el usuario lo pida explícitamente.

## Modo “movimiento constante” (WIP=1)

1. Si hay una tarea en **NOW**, avánzala 1 paso hoy (ejecución > planificación).
2. Si está bloqueada, pide **1 sola cosa exacta** para desbloquear.
3. Si NO hay NOW, crea 1 tarea de alto ROI y ponla en NOW.

## Coste bajo (obligatorio)

* Prioriza lo barato: checks rápidos + decisiones simples.
* Solo sube a trabajo “profundo” si: (a) hay bloqueo real, (b) hay deadline, (c) el ROI es obvio.
