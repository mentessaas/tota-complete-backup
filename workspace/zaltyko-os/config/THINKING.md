# Thinking Level Configuration

## Mi configuración
- **Thinking Level:** HIGH (cambiado el 2026-02-19)
- **Por qué:** Mejora la calidad de razonamiento
- **Nota:** MiniMax 2.5 soporta hasta "high"

## Cómo cambiarlo
- Por sesión: usar `/thinking high` en chat
- Para crons: configurarlo en el payload del cron
- Default: low → cambiar a high si no se especifica
