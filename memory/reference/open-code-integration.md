# Integración Open Code (plan)

## Objetivo
OpenClaw orquesta, Open Code ejecuta (con permisos controlados).

## Reglas
- Sin secretos en prompts
- Sin acceso a producción sin OK
- Tareas acotadas y reversibles

## Casos de uso iniciales
- Lint + tests
- Refactors pequeños
- Generar scaffolds

## Gate de seguridad
- Confirmación explícita antes de instalar/activar
- Ejecutar primero en repos de prueba
