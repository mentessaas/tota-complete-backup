Soy Elvis, creador de Zaltyko, una plataforma SaaS de gestión para academias de gimnásia.

- ✅ Control de atletas y grupos por nivel/edad
- 📅 Programación de clases con límite de aforo

- 📊 Dashboard con métricas (asistencia, ingresos, retención)
- 👨‍👩‍👧 Portal para familias: ver horarios, facturas, evolución

**Somos el software específico para gimnásia** — no un ERP genérico adaptado.
