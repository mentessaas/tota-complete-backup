## 14:12 - Análisis Simon Willison: Agentic Engineering Patterns

### Fuente
- https://simonwillison.net/guides/agentic-engineering-patterns/

### 6 Patrones Identificados
1. **Code is cheap, good code isn't** - Escribir código = barato, pero código bueno = caro
2. **Hoard patterns** - Acumular soluciones probadas con código funcional
3. **Red/Green TDD** - Tests primero → confirmar fail → implementar → pass
4. **First run tests** - Siempre ejecutar tests al inicio de sesión coding
5. **Linear walkthroughs** - Para entender código que no has escrito
6. **Interactive explanations** - Crear visualizaciones para conceptos complejos
